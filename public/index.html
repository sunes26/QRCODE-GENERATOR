QR코드란?

QR코드(Quick Response Code)는 2차원 바코드의 한 종류로, 스마트폰 카메라로 스캔하여 정보를 빠르게 읽을 수 있습니다.

주요 기능

  • 텍스트, URL, 이메일, 전화번호 QR코드 생성
  • WiFi 연결 정보 QR코드 생성
  • 명함(vCard) QR코드 생성
  • 위치 정보 QR코드 생성
  • 맞춤형 색상 및 로고 삽입
  • 다양한 바코드 형식 지원 (Code128, Code39, EAN13, UPC)
  • 대량 QR코드 일괄 생성
  • PNG, JPG, SVG, PDF 형식 다운로드
